# Автобиография Бенджамина Франклина

Книга находится в общественном достоянии.

Автор перевода неизвестен.

Книга была извлечена средствами OCR и Google Antigravity из DJVU файла, который был получен из открытых источников.

Главы были разделены на отдельные файлы, исправлены некоторые ошибки OCR и дополнены выводами от Gemini 3 Pro High, в целях лучшего понимания текста и извлечения пользы из прочитанного.
